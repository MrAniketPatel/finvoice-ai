# FinVoice.AI - Demo Setup Guide

## Quick Start (5 minutes)

### 1. Start the Backend
```bash
cd finvoice-backend
npm start
```
The backend will run on `http://localhost:5000`

### 2. Start the Frontend (in a new terminal)
```bash
cd finvoice-frontend
npm start
```
The frontend will open automatically at `http://localhost:3000`

## Demo Features to Showcase

### 1. Landing Page
- Beautiful storytelling design
- Animated floating cards
- Clear value proposition
- Smooth transitions and animations

### 2. Authentication
- Register a new account
- Login with credentials
- Secure JWT-based authentication

### 3. Dashboard
- Overview of financial stats (Income, Expense, Balance)
- Recent transactions display
- Upcoming payment alerts
- Clean, modern card-based design

### 4. Balance Sheet
- Add income/expense transactions
- Filter by time period (Week, Month, 6 Months, Year, All Time)
- View all transactions with delete option
- Real-time profit/loss calculation

### 5. Payable Alerts
- Create payment reminders (Rent, Loans, Bills)
- Set due dates and amounts
- Mark payments as paid
- Track pending vs paid alerts

### 6. Profile Management
- Update personal information
- Set occupation (Student, Employee, Business Owner)
- Add company details
