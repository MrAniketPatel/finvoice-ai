## If Something Goes Wrong

### Backend Crashes
1. Quickly restart: `npm start`
2. While restarting, explain the architecture
3. Show the code and models
4. Use backup screenshots

### Frontend Crashes
1. Hard refresh: Ctrl+Shift+R
2. If fails, show backend API
3. Explain React components
4. Use backup video/screenshots

### Complete Failure
1. Stay calm - this is a learning experience
2. Show the code and explain what it does
3. Walk through the architecture
4. Discuss challenges and solutions
5. Explain how you'd fix the issue
